# QLF Barber Shop - Full Stack Booking App

A premium, highly responsive booking application built with React, Vite, Tailwind CSS, Framer Motion, and Firebase. 

## Features
- **Real-Time Booking**: Uses Firebase `onSnapshot` to instantly gray out booked time slots for all visitors.
- **Telegram Bot Notifications**: The barber receives an instant message via Telegram the second a booking is confirmed.
- **Admin Dashboard**: A protected route (`/admin` - PIN `1234`) viewing all live upcoming appointments with a 1-click "Call Now" button.
- **Premium UI**: Crafted meticulously with Framer Motion animations and a dark-gold barber aesthetic.
